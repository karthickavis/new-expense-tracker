# Expense Tracker

A modern Expense Tracker app built using React and Tailwind CSS.

I built this app to track my daily expenses and improve my frontend development skills.

## Features

- Responsive Navbar
- Dark / Light Mode
- Mobile Hamburger Menu
- React Router Setup

## Tech Stack

- React
- Tailwind CSS
- Lucide React Icons
- React Router DOM
- React Toastify

## What I Learned

- Nested Routing
- React Router DOM
- Dark Mode using Tailwind CSS
- Responsive Design
- Navbar & Mobile Menu
